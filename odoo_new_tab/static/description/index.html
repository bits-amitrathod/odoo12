View Records In New Tab

This module allows user to open and view records of list in a new tab.

View Records In New Tab provides you the functionality to view each and every record in a new tab. It integrates a new tab icon beside every record thus, making it easier for a user to review records without running Odoo every time.

Now manage your time by reviewing records in different tabs in Odoo!!

It happens many times that a user has to maintain and verify multiple records at the backend. For that, selecting each tab and running Odoo browser everytime can make the task difficult and hectic too.

Instead, opening the record in a new tab is more feasible as you can get a clear approach for each and every record separately.Odoo view records in the new tab provide you a hassle-free approach to verify your records simultaneously in a single click.

Add a Record Tab button in odoo.

Now easily add a new tab button beside every record like sales order, purchase order, products, customers, invoice list. By clicking on this button, the user will be redirected to the new tab showing you the record details.

Detailed Features

Adds a new tab icon beside records in Odoo

  • This module adds a new tab icon beside every record in a list view.
  • Odoo users can click on the new tab button to open the corresponding record in a new tab without running Odoo every time.
  • Odoo view records in the new tab enable users to view multiple records in various new tabs simultaneously.

Directly close the record after review in Odoo

  • The module helps Odoo users in time management by eliminating the need for running Odoo every time to view a record.
  • It also enables users to directly close the record after reviewing it without closing the main Odoo tab.
