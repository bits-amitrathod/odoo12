SEO-URL Redirect/Rewrite

Better quality content and human readable keywords make more traffic to the website

Now make your website more SEO Friendly by rewriting your website URLs and make your website pages more searchable. This module also provides redirect URL feature to your website. Redirect feature helps to send an informative message to your user's browser which will tell them that the page is moved and it will automatically switch to a new page of your choice.

  • Features

    • Feature to redirect from source URL to destination URL on your website.
    • Feature to add multiple redirect list.
    • Feature to rewrite catalog product URL.
    • Feature to rewrite catalog product URL in bulk.
    • Feature to add a suffix for catalog product URL.
    • Feature to remove "/shop/product" from catalog product URL to make it SEO Friendly.
    • Feature to automatically redirect for catalog products.
    • Feature to rewrite product public category URL.
    • Feature to rewrite product public category URL in bulk.
    • Feature to add a suffix for product public category URL.
    • Feature to remove "/shop/category" from product public category URL to make it SEO Friendly.
    • Feature to automatically redirect for product public categories.
    • Feature to manage category hierarchy on Website.
    • Feature to manage category hierarchy along with product URL on Website.
    • Note: SEO-URL Redirect/Rewrite will only work with the Odoo default theme. To make it work with the customized theme or any third party theme you can contact us.

  • Catalog Rewrite Management

    • Feature to rewrite catalog product URL.

  • Feature to bulk rewrite catalog product/public categories URL

  • Catalog rewrite management on Website

  • Catalog Redirect Management

    • Feature to automatically redirect for catalog products/ public categories.

    • Feature to redirect from source url to destination url on your website.

  • Feature to add multiple redirect list

  • URL Suffix

    • Feature to add suffix for catalog product/public categories URL.

  • Manage Category Hierarchy

    • Feature to to manage category hierarchy on Website.

  • Manage Category Hierarchy Per Product

    • Feature to to manage category hierarchy along with product URL on Website.

  • Default Configuration

    • Enable suffix setting to add suffix along with URL.
    • User can add suffix for catalog product/ public categories as well.
    • Enable "use web server rewrite" setting to remove 'shop/category' and 'shop/product' from catalog URLs.
    • Enable "use category hierarchy" setting to manage category hierarchy on Website.
    • Enable "use category URL" setting to manage category hierarchy along with product URL on Website.
